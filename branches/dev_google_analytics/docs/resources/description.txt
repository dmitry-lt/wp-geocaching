Short description:
Russian: Клиент для geocaching.su
English: Geocaching.su mobile app


Detailed description:
Russian:
Геокэшинг - это мобильное приложение для работы с сайтом http://www.geocaching.su

Наше приложение упростит вам поиск тайников, с его помощью вы сможете:
- увидеть существующие тайники прямо на карте
- прочитать дополнительную информацию о тайнике
- сохранять тайники в избранное
- искать избранные тайники с помощью карты и компаса

Сайт проекта http://code.google.com/p/wp-geocaching/


Geocaching is a mobile client for the http://www.geocaching.su service. Please note, that most of the cache information is in Russian

This app will improve your geocaching experience by letting you
- view caches on a live map
- see detailed cache information
- add caches to favorites for further searching
- physically search favorite caches with the help of map and compass

Project site: http://code.google.com/p/wp-geocaching/
